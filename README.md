# EchoNova — Desktop Voice Assistant 🎙️

<img src="https://giffiles.alphacoders.com/212/212508.gif" alt="EchoNova Banner">

**EchoNova is a Python-based desktop voice assistant that helps you perform daily tasks using simple voice commands — no typing needed.**

Built with Python. Compiled in VS Code.

---

## 🛠️ Built With

![Python](https://raw.githubusercontent.com/github/explore/80688e429a7d4ef2fca1e82350fe8e3517d3494d/topics/python/python.png)

---

## ✨ Features

- 🕐 Tell current time and date
- 🌐 Open YouTube and Google
- 📖 Search Wikipedia and read results
- 🎵 Play music from your Music folder
- 😂 Tell jokes
- 📸 Take screenshots
- 💤 Shutdown / Restart your PC
- 🏷️ Change assistant name

---

## 📦 Installation

### Step 1 — Clone the repo
```bash
git clone https://github.com/your-username/echonova.git
cd echonova
```

### Step 2 — Install dependencies
```bash
pip install -r requirements.txt
```

### Step 3 — Run EchoNova
```bash
cd EchoNova
python echonova.py
```

---

## 🎤 Voice Commands

| Say this | EchoNova will |
|---|---|
| "what is the time" | Tell current time |
| "what is the date" | Tell current date |
| "open youtube" | Open YouTube |
| "open google" | Open Google |
| "wikipedia Python" | Search Python on Wikipedia |
| "play music" | Play a random song |
| "tell me a joke" | Tell a joke |
| "screenshot" | Take a screenshot |
| "change your name" | Rename EchoNova |
| "offline" or "exit" | Close EchoNova |
| "shutdown" | Shutdown your PC |
| "restart" | Restart your PC |
